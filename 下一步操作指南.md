# 高洲罗氏族谱小程序 - 下一步操作指南

## 你已经完成了
- ✅ 注册微信小程序
- ✅ 安装微信开发者工具
- ✅ 获取项目代码框架

## 接下来按顺序操作

### 步骤1：修改项目配置

打开 `project.config.json` 文件：
```json
{
  "appid": "你的小程序AppID"
}
```
把 `你的小程序AppID` 替换为你注册的小程序 AppID。

**如何找到 AppID：**
1. 登录 https://mp.weixin.qq.com
2. 进入"开发" → "开发管理" → "开发设置"
3. 复制"AppID(小程序ID)"

### 步骤2：创建项目

1. 打开微信开发者工具
2. 点击"+"（新建项目）
3. 填写：
   - 项目名称：`luozupu`
   - 目录：选择 `d:\DNS\luozupu\project` 文件夹
   - AppID：填写你的小程序 AppID
   - 后端服务：选择"微信云开发"
   - 模板：选择"不使用模板"
4. 点击"创建"

### 步骤3：开通云开发

1. 点击开发者工具上方的"云开发"按钮（云朵图标）
2. 点击"开通"（如果还没开通）
3. 选择"免费版"（默认就是免费版）
4. 记录"环境ID"（如：`luozupu-xxx`）
5. 修改 `miniprogram/app.js` 中的环境ID：
```javascript
wx.cloud.init({
  env: '你的环境ID',  // 替换这里
  traceUser: true
});
```

### 步骤4：创建数据库

1. 点击"云开发"按钮进入控制台
2. 点击"数据库"
3. 点击"添加集合"
4. 输入集合名称：`members`
5. 点击"确定"

### 步骤5：导入测试数据

**方法A：通过控制台手动导入（推荐）**

1. 在云开发控制台 → 数据库 → 选择 `members` 集合
2. 点击"添加记录"
3. 点击"导入JSON"
4. 复制 `database/init.js` 中的测试数据（去掉注释和 `module.exports =`）
5. 粘贴进去，点击"导入"

**方法B：使用云函数初始化**

1. 右键 `cloudfunctions/initDatabase` → "创建并部署"
2. 在控制台调用该云函数

### 步骤6：部署云函数

依次右键以下文件夹，选择"创建并部署：云端安装依赖"：
1. `cloudfunctions/getMembers`
2. `cloudfunctions/getMemberDetail`
3. `cloudfunctions/importMembers`

**部署成功标志：**
- 文件夹图标变成云朵形状
- 控制台显示"部署成功"

### 步骤7：运行测试

1. 点击开发者工具的"编译"按钮（或按 Ctrl+S）
2. 在模拟器中查看效果
3. 点击"预览" → 扫码在真机上测试

## 常见问题

### Q1：提示 "云开发环境未初始化"
**解决：** 检查 `app.js` 中的 `env` 是否填写正确

### Q2：云函数部署失败
**解决：** 
- 检查是否已开通云开发
- 检查网络连接
- 重试部署

### Q3：数据库查询为空
**解决：**
- 检查是否已创建 `members` 集合
- 检查是否已导入数据
- 检查查询条件是否正确

### Q4：页面显示空白
**解决：**
- 打开控制台查看错误信息
- 检查云函数是否部署成功
- 检查数据库权限设置
